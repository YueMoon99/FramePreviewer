### 📦 软件简介
**序列帧预览器**是一款专为动效设计师和 3D 艺术家打造的轻量级工具。它旨在解决预览序列帧时需要开启大型后期软件的痛点，提供丝滑、直观且开箱即用的预览体验。

### ✨ 核心功能
* **免安装单文件**：纯 Python + PyQt6 开发，打包后仅一个 exe 文件，随处运行。
* **智能序列识别**：拖入文件夹内**任意一帧**图片，软件将自动识别并加载同序列的所有文件，并自动适应窗口大小。
* **AE 风格交互**：
    * **RAM 缓存条**：实时显示已存入内存的帧（绿色进度条），确保高帧率播放不掉帧。
    * **播放头预览**：红色倒三角播放头，支持鼠标实时拖拽（Scrubbing）预览。
* **自由操控**：支持鼠标滚轮无级缩放、左键平移画面。
* **多维设置**：内置透明网格及多种纯色背景切换，预设 10-120 帧率选择。
* **极简 UI**：支持中英文一键切换，界面干净无广告。

### 🚀 如何使用
1.  下载`FramePrewiewer.exe`。
2.  将序列帧中的任意一张图片拖入软件窗口。
3.  按 **空格键** 播放/暂停。
4.  使用 **鼠标滚轮** 缩放，**左键拖拽** 图片查看细节，或在下方 **进度条** 拖动预览。

## English

### 📦 About
**Sequence Frame Previewer** is a lightweight tool designed specifically for motion designers and 3D artists. It eliminates the need to open heavy post-production software just to check a rendered sequence, providing a smooth, intuitive, and out-of-the-box experience.

### ✨ Key Features
* **Portable One-File**: Built with Python and PyQt6, compiled into a single executable for maximum portability.
* **Smart Detection**: Drag and drop **any single frame** from a folder; the app automatically identifies and loads the entire sequence.
* **AE-Inspired UI**:
    * **RAM Cache Bar**: Real-time display of cached frames (green bar), ensuring smooth playback even at high frame rates.
    * **Playhead Scrubbing**: A familiar red playhead for intuitive timeline scrubbing.
* **Flexible Navigation**: Supports infinite zooming with the mouse wheel and panning with left-click dragging.
* **Professional Settings**: Toggle between a transparent grid and multiple solid backgrounds; adjustable presets from 10 to 120 FPS.
* **Minimalist UI**: Bilingual support (CN/EN) with a clean, distraction-free interface.

### 🚀 Usage
1.  Download `FramePrewiewer.exe`.
2.  Drag and drop any image from your sequence into the window.
3.  Press **Space** to play/pause.
4.  Use the **Mouse Wheel** to zoom, **Left Click** to pan, or drag the **Timeline** to scrub through frames.
